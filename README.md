- Download and Install Windows 11.
-Follow the instructions to download and install the OS.

oRun the installer and follow the setup instructions.
3. Set up Version Control System
Step 1: Visit Git Download.
-Install Git and configure it on your local machine.
-Run the installer and follow the setup instructions.
Step 2: Visit GitHub and sign up.
-Create a GitHub account for hosting your repositories.
Step 3: Initialize a Git repository for your project and make your first commit.
-Open VS Code.
-Open the terminal 
-Run the following commands:
Git Bash
Git init
Git add .
Git commit -m "My first commit”
